# AliceInCradleOverpowered

[中文](README_cn.md) **English**

> Generated by GLM-5V-Turbo

An overpowered mod for AliceInCradle game.

## Requirements

- base game [AliceInCradle](https://get.aliceincradle.dev/win/)
- latest [BepInEx](https://github.com/BepInEx/BepInEx/releases/latest)

## Features

- Extend melee attack range
- Food won't be consumed on use
- Throwables won't be consumed on use
- Auto water shard shield during combat & explosive shards on hit
- Sort reel content by effect intensity
- Disable dynamic mosaic
- Suppress startup error log detection

## Deployment

1. Clone the repository
2. Copy `path.example.props` to `path.props` and replace `PATH/TO/GAME/ROOT` with your actual AliceInCradle game installation path
3. Build the project with your preferred IDE or build tool
