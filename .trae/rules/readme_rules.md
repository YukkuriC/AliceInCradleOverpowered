# README 更新规则

- 双语 `README.md` + `README_cn.md` 保持同步
- **Features/功能 采用多级缩进列表**，一级用 `**粗体**` 标注分类名，二三级通过缩进（2空格/4空格）体现层级，对应 `codegen/config.yaml` 的 `category` 字段（`/` 分隔表示层级）
- 每条基础条目（叶子节点）**严格对应 `config.yaml` 的一个 `name` 配置项**，不得按文件合并；基于该配置项的 `descrip` 翻译/改写，末尾标注 `` (`ConfigKey`) ``
- Features 列表与 `Patches/` 代码及 `codegen/config.yaml` 配置项一致，不得遗漏
- 署名：英文 `> Generated by <model>` / 中文 `> 生成于 <model>`
